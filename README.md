
# DIGITAL CLOCK
### A simple shell script project of Real-Time Clock.

- It shows the current local time.
- It's created using shell scripting langauge and run on the Terminal.


## Developement :
### Commands Used  -
- $ date 
- $ sleep 1s
- $ clear

 ### Other Expression Used -
  
  - ( while loop)
  - ( Color codes )
